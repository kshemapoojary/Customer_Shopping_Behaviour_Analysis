Overview

End-to-end data analytics project analyzing customer shopping behavior to identify spending patterns, customer segments, and subscription trends using Python, SQL, and Power BI.

Dataset

~3,900 customer purchase records

Includes demographics, purchase details, discounts, ratings, and shipping information

Tools

Python: Pandas

SQL: MySQL

Visualization: Power BI

Reporting: Gamma (PPT)

Workflow

Loaded and explored data in Python

Cleaned data and engineered new features

Performed business analysis using SQL in MySQL

Built an interactive Power BI dashboard

Created a report and presentation summarizing insights

Dashboard

Power BI dashboard with KPIs, customer segmentation, revenue analysis, and interactive slicers.

Key Insights

Subscribers and repeat buyers generate higher revenue

Certain products depend heavily on discounts

Revenue varies significantly by age group and shipping type

How to Run

Run Python scripts for data cleaning and EDA

Load cleaned data into MySQL for SQL analysis
