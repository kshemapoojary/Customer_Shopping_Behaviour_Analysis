# Customer Shopping Behaviour Analysis

## Overview
End-to-end data analytics project analyzing customer shopping behavior to identify spending patterns, customer segments, and subscription trends using Python, SQL, and Power BI.

## Dataset
- ~3,900 customer purchase records  
- Demographics, purchase details, discounts, ratings, and shipping information  

## Tools
- Python (Pandas)
- MySQL (SQL Analysis)
- Power BI (Dashboard)
- Gamma (Presentation)

## Workflow
1. Loaded and explored data using Python  
2. Cleaned data and created new features  
3. Performed business analysis using SQL in MySQL 
4. Built an interactive Power BI dashboard  
5. Created report and presentation  

## Dashboard
Interactive Power BI dashboard with KPIs, customer segmentation, revenue insights, and slicers.

## Key Insights
- Subscribers and repeat buyers generate higher revenue  
- Some products rely heavily on discounts  
- Revenue varies by age group and shipping type  

## How to Run
- Run Python scripts for data cleaning and EDA  
- Load cleaned data into MySQL for SQL Analysis 
- Open Power BI file to explore the dashboard
